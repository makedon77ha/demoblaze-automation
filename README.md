# DemoBlaze Automation Project

## Overview
Automation framework for testing DemoBlaze website using Selenium, TestNG and Maven.

## Technologies
- Java 17
- Selenium WebDriver
- TestNG
- Maven

## Framework Features
- Page Object Model (POM)
- Explicit waits (WebDriverWait)
- Screenshot on failure
- TestNG Listener
- Parallel test execution

## Test Coverage
- Signup tests
- Login tests
- Product tests
- Checkout tests

## Running Tests
Run tests using testng.xml

## Reports
Reports are generated in:
test-output/ExtentReport.html

## Notes
